My learning

Tugas Programmer Mengembangkan program yang sudah didesain dengan pilihan bahasa pemrograman, semisal Python, SQL, dan Javascript. Menguji (testing) fitur-fitur dalam program. Melakukan deploy atau mengimplementasikan program komputer ke semua target perangkat mulai dari mobile sampai desktop.

Apa yg dimaksud dengan bahasa pemrograman? Bahasa pemrograman, atau sering diistilahkan juga dengan bahasa komputer atau bahasa pemrograman komputer, adalah instruksi standar untuk memerintah komputer. Bahasa pemrograman ini merupakan suatu himpunan dari aturan sintaks dan semantik yang dipakai untuk mendefinisikan program komputer.

Teknik Informatika menawarkan berbagai peluang karier di berbagai industri. Lulusan dapat bekerja dalam berbagai peran seperti insinyur perangkat lunak, pengembang web, administrator jaringan, analis sistem, analis data, administrator basis data, dan profesional keamanan siber
